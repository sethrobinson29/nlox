# nlox

Working through [Crafting Interpreters](https://craftinginterpreters.com/) by Robert Nystrom, implemented in Nim.

## About

Lox is a dynamically typed scripting language built across the book. This implementation covers both the tree-walk interpreter (jlox) and bytecode virtual machine (clox) from the book, with idiomatic adjustments for Nim.

## Status

Work in progress, following the book chapter by chapter.
